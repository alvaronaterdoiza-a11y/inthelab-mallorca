# In The Lab Mallorca — Gestor de bonos

App web para gestionar bonos de tecnificación de baloncesto: asistencia, sesiones restantes y pagos. Conectada a Supabase como base de datos.

## Desarrollo local

```
npm install
npm run dev
```

## Despliegue

Pensada para desplegar en Vercel conectando este repositorio de GitHub directamente.
